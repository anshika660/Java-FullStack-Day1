##  Overview

This repository contains the **Day 1 assignment** completed as part of the **TalentxMinds Java Full Stack Development Program**.

The objective of this assignment was to set up the Java development environment, understand the Java execution process, and create the first Java application while learning the basics of Git and GitHub.

---

##  Learning Objectives

- Understand Java fundamentals
- Install and configure JDK
- Learn the difference between JDK, JRE, and JVM
- Write and execute a Java program
- Initialize a Git repository
- Push code to GitHub
- Learn the basic developer workflow

##  Getting Started

### Clone the repository

```bash
git clone https://github.com/anshika660/Java-FullStack-Day1.git
```

### Navigate to the project

```bash
cd Java-FullStack-Day1/src
```

### Compile

```bash
javac Main.java
```

### Run

```bash
java Main
```
