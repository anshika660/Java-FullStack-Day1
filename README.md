## 📂 How to Run

Clone the repository

```bash
git clone https://github.com/anshika660/Java-FullStack-Day1.git
```

Navigate to the source folder

```bash
cd Java-FullStack-Day1/src
```

Compile

```bash
javac Main.java
```

Run

```bash
java Main
```
